WP Hooker records all fired WP hooks and their hooked functions, to easily let you debug hook issues!

== Description ==

Are you tearing your hair out because that function isn't firing, even though you hooked it?
Do you want to find the thing that overrides your changes?
Want to know why that thing does what it does?
Ever wondered what mysteries lies beyond the WordPress interface?

Let WP Hooker help you!

By recording all fired hooks and their functions during each page request, WP Hooker helps you with what otherwise would be hard to see. 

== Installation ==

WP Hooker should only be used in a Development environment. 

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to 'WP Hooker Options', in the admin area, and switch 'Unleash the Hooker' to 'On' (Remember to save)
4. Start debugging!
5. View your recorded sessions under 'Hooker Sessions'
